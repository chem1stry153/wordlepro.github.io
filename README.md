# Xcode Usage

1. Open Xcode and create a new iOS App project (SwiftUI).
2. Delete the default `YourAppNameApp.swift` and `ContentView.swift`.
3. Add `WordlesProPromoApp.swift` from this folder into your Xcode project.
4. Make sure the file is added to your app target.
5. Run on Simulator or open SwiftUI Preview.

This view is designed as an App Store style promo screen for `wordles.pro`, including animated letter states:

- Green: correct letter and correct position
- Yellow: letter exists but misplaced
- Red: letter not in target word
